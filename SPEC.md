# 保险佣金计算器 - 需求规范

## 1. 项目概述

- **项目名称**: InsuranceCommissionPal
- **Bundle Identifier**: com.insurance.commissionpal
- **核心功能**: 保险代理人可通过选择保险公司、保险产品，输入首年保费，快速计算税前首年佣金
- **目标用户**: 保险代理人
- **iOS版本支持**: iOS 15.0+

## 2. UI/UX 规范

### 屏幕结构
- **单一页面设计**: 主屏幕包含所有输入和计算结果显示
- **导航**: 无需导航栏，简单直接的单页应用

### 视觉设计

#### 颜色
- **主色调**: #2563EB (蓝色 - 专业信任)
- **次要色**: #10B981 (绿色 - 成功/佣金)
- **背景色**: #F8FAFC (浅灰白)
- **文字主色**: #1E293B (深灰)
- **文字次要色**: #64748B (中灰)
- **边框色**: #E2E8F0 (浅灰)
- **错误色**: #EF4444 (红色)

#### 字体
- **标题**: System Bold, 24pt
- **副标题**: System Semibold, 18pt
- **正文**: System Regular, 16pt
- **标签**: System Medium, 14pt
- **结果显示**: System Bold, 32pt

#### 间距
- **屏幕边距**: 16pt
- **元素间距**: 12pt
- **卡片内边距**: 16pt
- **圆角**: 12pt

### 视图组件

1. **App标题**: 顶部居中，显示"保险佣金计算器"

2. **保险公司选择器**
   - 下拉选择框 (Picker)
   - 选项：中国人寿、平安保险、太平洋保险、友邦保险、太平人寿、新华保险
   - 默认提示："请选择保险公司"

3. **保险产品选择器**
   - 下拉选择框 (Picker)
   - 根据选择的保险公司动态显示对应的产品
   - 默认提示："请先选择保险公司"
   - 选中公司后显示该公司的产品列表

4. **首年保费输入框**
   - UITextField
   - 键盘类型: 数字键盘
   - 带¥符号前缀
   - 占位符: "请输入首年保费"

5. **计算按钮**
   - 蓝色背景 (#2563EB)，白色文字
   - 圆角按钮
   - 文字："计算佣金"

6. **结果显示区域**
   - 卡片样式，白色背景
   - 显示：税前首年佣金金额
   - 绿色大字号突出显示

## 3. 功能规范

### 核心功能

#### 保险公司与产品数据
```
中国人寿:
  - 国寿福臻享版 (佣金率 35%)
  - 国寿康宁 (佣金率 40%)
  - 国寿鑫享鸿福 (佣金率 30%)

平安保险:
  - 平安福2019 (佣金率 38%)
  - 平安守护符 (佣金率 35%)
  - 平安智盈人生 (佣金率 32%)

太平洋保险:
  - 金福人生 (佣金率 36%)
  - 安心保 (佣金率 33%)
  - 人生启航 (佣金率 30%)

友邦保险:
  - 全佑至珍 (佣金率 42%)
  - 全佑倍护 (佣金率 40%)
  - 稳赢一生 (佣金率 35%)

太平人寿:
  - 福禄康瑞 (佣金率 37%)
  - 卓越人生 (佣金率 34%)
  - 悦享健康 (佣金率 32%)

新华保险:
  - 健康无忧C (佣金率 38%)
  - 多倍保 (佣金率 35%)
  - 返还型定期 (佣金率 30%)
```

#### 佣金计算逻辑
- **佣金 = 首年保费 × 佣金率**
- 结果四舍五入到元

### 用户交互流程
1. 用户打开App
2. 选择保险公司 → 产品列表自动更新
3. 选择保险产品
4. 输入首年保费
5. 点击"计算佣金"按钮
6. 显示计算结果

### 边界情况处理
- 未选择保险公司: 提示"请选择保险公司"
- 未选择产品: 提示"请选择保险产品"
- 未输入保费或输入0: 提示"请输入首年保费"
- 输入非法字符: 自动过滤非数字字符

## 4. 技术规范

- **UI框架**: SwiftUI
- **项目结构**: 单一视图文件
- **数据存储**: 无需持久化 (内存中处理)
- **依赖**: 无第三方依赖
